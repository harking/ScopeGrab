Simple serial console

The program has a simple serial trace and debug command console. Before or after connecting, you can switch to the "Serial console" tab. In the textbox below, you can write Fluke commands (like ID for ident, RI for reset instrument, GD for power down). Click "Send" to execute.


A full list of available Fluke commands should be in your Scopemeter service manual or the Remote Control and Programming Reference for the Fluke 190 family. One copy of the 190 reference is at http://www.iki.fi/jan.wagner/fluke/Fluke-RS232-reference.pdf.
